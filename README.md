# JavaScript Cart Total Calculator

A beginner JavaScript assignment that calculates the total cost of products in a shopping cart.

## Features

- Store product prices in an array
- Calculate the total cart value
- Display the final bill
- Apply a **3% discount** to each product (Bonus)

## Example

```text
Input:
[1200, 500, 800]

Output:
2500
```

## Technologies Used

- HTML5
- JavaScript

## JavaScript Concepts Used

- Arrays
- Variables
- Loops
- Arithmetic operators
- Addition
- Percentage calculation

## Purpose

This project was created to practice working with arrays, loops, arithmetic operations, and calculating the total cost of products in a shopping cart using JavaScript.
